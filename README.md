# Ekstrasi Fitur Warna Pixel HSV pada Buah Strawberry

## Dataset
Dataset dapat diakses melalui link berikut:
https://drive.google.com/drive/folders/1PxEQ6LqHjF-6uRpepyAAtkvVATQQZbAf?usp=sharing 

## Deskripsi
Project ini bertujuan untuk mengekstraksi fitur warna HSV dan tekstur GLCM dari citra buah stroberi untuk menganalisis tingkat kematangan.

## Tools
- Python
- OpenCV
- NumPy
- scikit-image

## Cara Menjalankan
1. Download dataset dari Google Drive
2. Jalankan file Python
3. Hasil ekstraksi akan ditampilkan dalam bentuk nilai HSV dan GLCM

## Hasil
Menunjukkan variasi kematangan berdasarkan nilai Saturation dan tekstur berdasarkan Contrast.
